Questions de cours (6 points)

  1. Définition d'un cycle de Sn : Un cycle de Sn est une permutation de l'ensemble {1, ..., n} qui peut être représentée comme un produit de transpositions. Plus précisément, un cycle de longueur k est une permutation qui peut être représentée comme (a1 a2 ... ak), où a1, a2, ..., ak sont des éléments distincts de {1, ..., n}. Par exemple, (1 2 3) est un cycle de longueur 3 dans S3.

  2. Forme p-linéaire alternée : Une forme p-linéaire alternée sur un espace vectoriel E est une application multilinéaire φ : E^p → R telle que φ(x1, ..., xp) = 0 si deux des arguments xi et xj sont égaux. De plus, φ est antisymétrique, c'est-à-dire que φ(x1, ..., xp) = (-1)^(i+j) φ(x1, ..., xi-1, xj, xi, ..., xj-1, xi, xj+1, ..., xp) si i < j.

Pour montrer qu'une forme p-linéaire alternée est antisymétrique, on peut utiliser la définition de l'antisymétrie. Soit φ une forme p-linéaire alternée et x1, ..., xp des vecteurs de E. On a :

φ(x1, ..., xp) = φ(x1, ..., xi-1, xi, ..., xj-1, xj, xj+1, ..., xp)

En échangeant les arguments xi et xj, on obtient :

φ(x1, ..., xi-1, xj, xi, ..., xj-1, xi, xj+1, ..., xp) = (-1)^(i+j) φ(x1, ..., xi-1, xi, ..., xj-1, xj, xj+1, ..., xp)

Puisque φ est alternée, on a :

φ(x1, ..., xi-1, xj, xi, ..., xj-1, xi, xj+1, ..., xp) = 0

Donc, φ(x1, ..., xp) = (-1)^(i+j) φ(x1, ..., xi-1, xi, ..., xj-1, xj, xj+1, ..., xp) = 0

Cela montre que φ est antisymétrique.

  1. Définition de exp(A) : Soit A une matrice carrée de taille n × n à coefficients complexes. La fonction exponentielle de A est définie comme :

exp(A) = ∑[k=0 à ∞] (A^k) / k!

où A^k est la puissance k-ième de A et k! est la factorielle de k.

  1. Commutation de P(u) et Q(u) : Soient P et Q deux polynômes dans R[X] et u un endomorphisme d'un espace vectoriel E. On a :

P(u) Q(u) = (∑[k=0 à n] a_k u^k) (∑[l=0 à m] b_l u^l)

En utilisant la propriété de commutation des puissances de u, on obtient :

P(u) Q(u) = ∑[k=0 à n] ∑[l=0 à m] a_k b_l u^(k+l)

De même, on a :

Q(u) P(u) = ∑[l=0 à m] ∑[k=0 à n] b_l a_k u^(l+k)

Puisque k+l = l+k, on a :

P(u) Q(u) = Q(u) P(u)

Cela montre que P(u) et Q(u) commutent.
